# CommonChat 组件

抽取一些echarts图表的公共ui，方便在其他页面中使用。

## 代码演示

```tsx
import React from 'react';
import CommonChat from '@/components/CommonChat';

export default () => <CommonChat  >公共图表</CommonChat>;

```
